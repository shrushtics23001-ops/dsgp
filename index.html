How to Play

Objective

Transform the initial data structure state to match the target state using the allowed operations.

Controls

  • Select a data structure from the main menu
  • Choose a level based on difficulty
  • Use operation buttons to modify the data structure
  • Complete the puzzle in minimum moves

Scoring

  • Correct moves: +10 points
  • Level completion: +50 points
  • Time bonus: Up to +30 points
  • Hint usage: -5 points

Data Structures

  • Stack: LIFO (Last In, First Out)
  • Queue: FIFO (First In, First Out)
  • Linked List: Dynamic insertion/deletion
  • Tree: Hierarchical organization
  • Graph: Network of connected nodes
